Read me

1.Imagepuzzle.h
- is the header file and contains all the methods that both the Imagepuzzle.cpp and Imagepuzzle.cpp file will require
2. Imagepuzzle.cpp
- contains the implementation of all the methods in the Imagepuzzle header
3.Driver.cpp
- Contain the main method that read the file and call the method in the Imagepuzzle.cpp to perform a required action
The main method can only make correct output for all the cases mastery problem

To run the program: ProgramName -s (sizeOfGrid) -n (numberOfMoves) -i (InputImage) -x(outputImageName)
e.g  ./puzzle 100 20 mona512 summary_out
